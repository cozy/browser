A custom field label must :

  • Either start by :
    • 'regex='
    • 'csv='
    and the expression will be evaluated againts the following field's attributes : htmlID, htmlName, label-tag, label-aria, placeholder, label-left. (see `fieldPropertyIsMatch` in autofil.service.ts)

  • or exactly match the content of :
    • htmlID
    • htmlName
    • label-tag
    • label-aria
    • placeholder
    • label-left
