❗❗ Note : autofil will be triggered in all frames. That is a choice. The reson is that limitating the autofill to only the source frame is possible only when trigered from the menu. This is not possible when autofill is launched from the popup since this it has no idea of wich frame to target.
So we send the message to all frames of a specific tab, assuming all frames have the same level of trust than the tab url.
